# NumPy Masterclass (From Scratch to Advanced in 2 Days)

This repository walks through NumPy step-by-step with clear explanations and practical examples.  
It's designed for Data Science and Machine Learning learners who want fast, solid mastery.

**What this repo covers:**
- NumPy fundamentals
- Array creation & manipulation
- Mathematical operations
- Statistics & aggregations
- Indexing, slicing, and filtering
- Broadcasting & reshaping
- Real-world data handling

**Environment:**
- Python 3.11+
- NumPy latest version
- Jupyter / VS Code

Each folder contains `.ipynb` notebooks + markdown notes.

Learn → Practice → Commit → Repeat.
